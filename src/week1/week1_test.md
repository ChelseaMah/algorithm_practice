# 第一周
1. 一个顾客购买了价值x元的商品（不考虑角、分），并将y元的钱交给售货员，编写算法，在各种币值的钱都很充足的情况下，使售货员使用张数最少的钱币找给顾客。
    >分析：无论买商品的价值x是多少，找给他的最多需要以下6种币值：50，20，10，5，2，1。
